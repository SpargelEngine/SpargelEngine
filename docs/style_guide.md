# Spargel Style Guide

## C++

- Do not use RTTI or exceptions.
- Avoid the standard library.
- Use `#pragma once`.

### Naming Convention

- File: `some_file.h`, `some_file.cc`
- Namespace: `some_namespace`
- Class: `SomeClass`
- Function: `someFunction`
- Variable: `some_variable`
- Private variable: `_private_variable`
